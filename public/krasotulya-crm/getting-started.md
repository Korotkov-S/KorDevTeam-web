# Как начать работать в Красотуля-CRM

Ниже — быстрый старт, чтобы за 10–15 минут получить рабочую CRM.

## Шаг 1. Подготовьте сущности

- Добавьте ответственных пользователей
- Создайте этапы воронки
- Определите обязательные поля карточки сделки

## Шаг 2. Заведите первых клиентов

1. Создайте контакт
2. Привяжите компанию (если нужно)
3. Создайте сделку и поставьте задачу на следующий шаг

## Шаг 3. Настройте дисциплину работы

- Любая коммуникация фиксируется в карточке
- Нет задачи — нет следующего шага
- Еженедельный разбор воронки по статусам
