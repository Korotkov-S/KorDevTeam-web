# Getting started with Krasotulya-CRM

This is a quick start to get a working CRM in 10–15 minutes.

## Step 1. Prepare entities

- Add responsible users
- Create pipeline stages
- Define required fields for a deal card

## Step 2. Add your first customers

1. Create a contact
2. Link a company (optional)
3. Create a deal and add a task for the next step

## Step 3. Establish operating rules

- Log every communication in the card
- No task — no next step
- Weekly pipeline review by stages/statuses
